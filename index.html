How this player works

This is a 100% static, open-source page. It pulls the live iptv-org playlist (~12,500 streams) straight from their public API in your browser and plays HLS with hls.js. No server, no tracking, nothing stored remotely.

Categorisation

Channels are joined to iptv-org metadata and grouped by category (News, Sports, Movies, Music, Kids…), country (with flag), language, and quality. Use the dropdowns and search to slice all of it.

Status badges

GeoRestrictedOKCORSOfflineNote
  • Geo — iptv-org flags this stream as geo-blocked (label in the data). It will likely fail unless you're in the right country.
  • Restricted — the stream requires a custom Referer or User-Agent header. Browsers forbid setting those on media requests, so it can't play here. Works in VLC/mpv or via a proxy.
  • CORS — the stream's server sends no cross-origin headers, so the browser blocks it. Needs a proxy (below).
  • OK — verified playing in this browser. Offline — source is dead/unreachable.

Geo & Restricted are read straight from the data. CORS / OK / Offline are determined live the moment you click a channel, then cached.

Why some channels won't play (and the fix)

A static page can only play streams that (a) allow cross-origin requests and (b) aren't geo-locked from your location. Chrome/Firefox are strict; Safari plays HLS natively and bypasses CORS, so it works for far more channels. To unlock everything, route streams through a tiny proxy — e.g. a Cloudflare Worker that adds CORS headers and the required Referer/User-Agent. Drop its URL in and every stream becomes playable.
